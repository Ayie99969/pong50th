# pong50th
A pong game celebrating 50th anniversary, made with raylib-rs

## Controls
### Keyboard
- [W] - Up
- [S] - Down

### Gamepad
- Left Stick Up - Up
- Left Stick Down - Down

### D-Pad
- Up - Up
- Down - Down
